Logic CPU system

Features include:

  • 4 word ROM size - 6 bits each
  • Powered by Mini-USB
  • 4-bit accumulator
  • Manual pulse generator with de-bounce circuit
  • Comes in a stylish box

Opcodes

  1. LOAD - acc:=v - Load a value into accululator
  2. ADD - acc:=acc+v - Add constant into accululator
  3. LSL -  acc:=<<acc - Logically shift accumulator bits left by 1
  4. JMP - PC:=v - Sets PC to value in ROM
